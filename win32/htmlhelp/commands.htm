Editor Commands

Commands are entered as - [p1][,p2][,p3][,p4][?]command[argument]

A line number on its own edits that single line.


1 2 3 4 option command argument comment
A append from input
first last dest count C copy lines
first last D delete lines
E save file and exit
line I insert, end with ^Z
first last L list
first last dest M move lines
first last P page
Q quit without saving
first last ? R search^Zreplace replace text
first last ? S search search for text
line T filename merge text from file
count W write to output file
